# Playwright Test for Events Widget

Этот репозиторий содержит автоматизированный тест для страницы [Events Widget (3Snet)](https://dev.3snet.info/eventswidget/) с использованием [Playwright](https://playwright.dev/) и TypeScript. 

Тест спроектирован с учетом паттерна Page Object Model (POM) для модульности и простоты расширения.

## 📌 Что проверяется в тесте?

В рамках тестового задания реализованы следующие проверки:
1. **Базовая загрузка страницы**: проверка корректности HTTP-ответа и проверка того, что Title страницы содержит текст `Конструктор календаря мероприятий`.
2. **Отображение ключевых элементов**: проверка видимости главного заголовка H1 `Начните создавать свой календарь мероприятий!`.
3. **Наличие необходимых ссылок**: проверка наличия ссылки на `календарь мероприятий`, которая необходима для корректного флоу пользователя.

*Архитектура:* 
- `pages/EventsWidgetPage.ts` — описание страницы (Page Object Model), локаторы и методы взаимодействия.
- `tests/eventswidget.spec.ts` — непосредственно тестовые сценарии.
- `playwright.config.ts` — конфигурация Playwright для запуска тестов в параллельном режиме в браузерах Chromium, Firefox и WebKit.

## 🚀 Как запустить локально?

Проект является самодостаточным. Для его запуска на вашем компьютере должен быть установлен [Node.js](https://nodejs.org/) (рекомендуется версия 18 или новее).

1. **Клонируйте репозиторий:**
   ```bash
   git clone <URL_репозитория>
   cd <Имя_папки_репозитория>
   ```

2. **Установите зависимости:**
   ```bash
   npm install
   ```
   *(Также необходимо установить браузеры Playwright, если они не установлены: `npx playwright install --with-deps`)*

3. **Запустите тесты:**
   - Выполнить тесты в headless-режиме (без UI):
     ```bash
     npm test
     ```
   - Выполнить тесты с открытием браузера:
     ```bash
     npm run test:headed
     ```
   - Запустить Playwright UI режим:
     ```bash
     npm run test:ui
     ```

## 📊 Результаты выполнения

После прохождения тестов Playwright автоматически сгенерирует HTML-отчет (если в конфигурации указан html-репорт). Вы можете открыть его командой:
```bash
npx playwright show-report
```

В проекте также настроен GitHub Actions workflow (`.github/workflows/playwright.yml`), который будет автоматически запускать тесты при каждом Push или Pull Request в ветку `main`.
