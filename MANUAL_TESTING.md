# Ручное тестирование виджета (Test Cases)

Здесь описано, как вручную пройти те же самые проверки, которые выполняет наш автоматизированный тест.

## Сценарий 1: Проверка локализации (Смена языка)
1. Откройте страницу https://dev.3snet.info/eventswidget/
2. **Ожидаемый результат**: Заголовок страницы содержит "Конструктор календаря мероприятий".
3. Найдите переключатель языка (вверху справа или в меню) и выберите **English (EN)**.
4. **Ожидаемый результат**: URL страницы поменяется на `.../en/eventswidget/`, а заголовок переведется на "Events calendar".

## Сценарий 2: Отсутствие JS ошибок в консоли
1. Обновите страницу конструктора.
2. Нажмите `F12` (Developer Tools) и откройте вкладку **Console** (Консоль).
3. Измените ширину и высоту на 600x400, поставьте галочку "На ширину контейнера".
4. Нажмите кнопку **"Сгенерировать превью"**.
5. Посмотрите на появившееся снизу превью (iframe).
6. **Ожидаемый результат**: Во время всех этих действий в консоли не должно появляться никаких новых красных ошибок.

## Сценарий 3: Проверка подстановки тематик (API/Network Interception)
1. Откройте дропдаун **"Выберите тематику"**.
2. Уберите галочку "Выбрать все" (если она стоит) и выделите конкретно **"Affiliate"** и **"SEO"**.
3. Нажмите кнопку **"Сгенерировать превью"**.
4. Посмотрите на сгенерированный код вставки (в текстовом поле).
5. **Ожидаемый результат**: В ссылке `src="..."` для iframe внутри кода должны появиться параметры ID выбранных тем. Например: `...&event_type=...` (не должно быть пустым или просто "on", если выбраны конкретные). 

## Сценарий 4: Выбор цветовых тем
1. Найдите блок выбора цвета (радио-кнопки зеленый, синий, фиолетовый и темно-зеленый/чирок).
2. Поочередно выбирайте каждый из цветов.
3. Проверяйте сгенерированный код внизу (для этого возможно придется еще раз нажимать Сгенерировать превью в зависимости от реализации сайта, либо код меняется на лету).
4. **Ожидаемый результат**: В сгенерированном текстовом коде параметр темы `theme=...` моментально меняется на соответствующий выбранному цвету (green, blue, turquoise, purple).

## Сценарий 5: Работа буфера обмена (Copy Clipboard)
1. Нажмите "Сгенерировать превью", чтобы в текстовом поле появился код.
2. Нажмите на кнопку **"Скопировать код"** (или просто "Скопировать").
3. Откройте любой текстовый редактор (Блокнот) и нажмите `Ctrl + V` (Вставить).
4. **Ожидаемый результат**: Вставленный текст должен абсолютно точно совпадать с текстом из текстового поля `textarea` на странице. Однотипные спецсимволы вроде `&` или угловых скобок `<` не должны биться.
